# Festa

## 로그인 페이지
- 아이디값 입력시 아이디 존재 여부를 분기처리하여 해당 로그인/회원가입 폼 view 구현

## 메인 페이지
- 알림 설정 팝업
- 무료/유료/외부 이벤트 카테고리 분류하여 노출
- 이벤트 클릭시 이벤트 상세 정보 노출
- 자세히보기 클릭시 해당 이벤트 상세페이지로 라우팅


사용 라이브러리
- react-router-dom
- react-error-boundary
- styled-components
- uuid
- axios
- react-slick
- slick-carousel
- query-string
